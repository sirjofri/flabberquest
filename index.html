Der Hintergrund

Hier wird etwas Lore beschrieben. Dies ist für den Spielverlauf jedoch nicht entscheidend, erklärt jedoch, warum die Spieler als "Stimme" bezeichnet werden.

Die Spielregeln

Spielbeginn: Der beginnende Spieler wird per Zufall bestimmt. Er wählt ein Feld auf dem Spielfeld als Mauer aus. Anschließend wählt der andere Spieler ebenfalls ein Spielfeld als Mauer aus. Nachdem der erste Spieler ein drittes Feld als Mauer ausgewählt hat, bestimmt der zweite Spieler die Ausgangsposition, indem er die Spielfigur auf ein Feld setzt.

Spielablauf: Die Spieler ziehen abwechselnd eine zufällig bestimmte Anzahl an Feldern. Sie versuchen, möglichst im Umfang der Augenzahl in eine Richtung zu gehen (diagonal oder gerade). Dabei ist eine Mauer nicht betretbar. Wird ein Spieler vom Spielfeldrand oder von einer Wand aufgehalten, wird die übrigbleibende Augenzahl als Strafpunkte notiert. Der Spieler, der zuerst eine Strafpunktzahl von 10 erreicht, gilt als Verlierer.
